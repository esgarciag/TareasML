# Tareas ML

En este repositorio están las **tareas dejadas en clase, ya desarrolladas** (demostraciones y análisis), en la carpeta `soluciones`.

Además hay unos **notebooks** con las implementaciones en código (SVM y perceptrón, desde cero).

## Cómo correr los notebooks

Necesitas Python 3 y estas librerías:

```
pip install numpy matplotlib
```

Luego ábrelos en VS Code y dale a **Run All**.
