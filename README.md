# Tareas ML

En este repositorio están las **tareas dejadas en clase, ya desarrolladas** (demostraciones y análisis), en la carpeta `soluciones`.

Además hay unos **notebooks** con las implementaciones en código.

## Cómo correr los notebooks

Se necesita Python 3 y estas librerías:

```
pip install numpy matplotlib
```

Luego se abre en VS Code y se corre todo.
